# 사용자 가이드: 새 기기 추가 및 설정 변경 방법

이 시스템을 다른 환경(노트북, 새로운 핸드폰)에서 사용하거나, 감지 대상을 변경할 때 수정해야 할 부분을 정리했습니다.

## 1. 감지 대상(Trusted Devices) 추가하기
가족 구성원이 늘어나거나, 노트북도 재실 감지 대상에 포함시키고 싶다면 `config.json`을 수정해야 합니다.

### [config.json](./config.json) 수정
`trusted_devices` 목록에 새로운 기기의 **MAC 주소**를 추가하세요.

```json
{
  "trusted_devices": [
    "C2:77:BE:99:90:A1",  // 기존 폰
    "AA:BB:CC:DD:EE:FF"   // [추가] 새로운 폰 또는 노트북
  ]
}
```

> [!IMPORTANT]
> **필수 설정**: 추가하려는 기기(폰/노트북)의 와이파이 설정에서 **"랜덤 MAC(비공개 주소)" 기능을 꺼야 합니다.** 그렇지 않으면 주소가 계속 바뀌어 인식이 불가능합니다.

## 2. 네트워크 반경이 바뀌었을 때 (이사, 공유기 교체)
시스템을 다른 장소로 옮겼다면 네트워크 설정이 바뀝니다.

### [config.json](./config.json) 수정
1.  **MQTT 호스트 (`mqtt.host`)**:
    *   MQTT 브로커(서버) 역할을 하는 PC나 파이의 **새로운 IP 주소**로 바꿔주세요.
2.  **네트워크 인터페이스 (`system.arp_interface`)**:
    *   오렌지파이/리눅스 기기마다 와이파이 장치 이름이 다를 수 있습니다. (예: `wlan0`, `wlP2p33s0` 등)
    *   터미널에서 `ifconfig` 또는 `ip a`를 입력해 현재 쓰고 있는 인터페이스 이름을 확인 후 수정하세요.
