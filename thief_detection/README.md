# 강도 침입 감지 시스템 (Thief Detection System)

## 프로젝트 개요
온디바이스 AI 기반 강도 침입 감지 시스템입니다. ARP 스캔을 통해 사용자 기기의 존재 여부를 확인하고, YOLOv8을 사용한 실시간 객체 인식으로 사람을 감지하여 침입 여부를 판단합니다.

## 주요 기능
1. **ARP 스캔**: 동일 네트워크 상의 기기를 스캔하여 사용자 기기 존재 확인
2. **YOLOv8 객체 인식**: 웹캠을 통한 실시간 사람 감지
3. **침입 판단**: 사용자 부재 시 사람 감지되면 경보 발생

## 시스템 요구사항
- Python 3.8 이상
- 웹캠
- 동일 네트워크에 연결된 환경
- Windows/Linux/MacOS

## 설치 방법
```bash
pip install -r requirements.txt
```

## 사용 방법

### 1. 설정 파일 수정
`config.json` 파일에서 사용자 기기의 MAC 주소를 설정합니다:
```json
{
  "trusted_devices": [
    "AA:BB:CC:DD:EE:FF",
    "11:22:33:44:55:66"
  ],
  "network_range": "192.168.0.0/24",
  "detection_threshold": 0.5,
  "alert_cooldown": 30
}
```

### 2. 시스템 실행
```bash
python main.py
```

## 파일 구조
- `arp_scanner.py`: ARP 스캔 모듈
- `person_detector.py`: YOLOv8 기반 객체 인식 모듈
- `main.py`: 메인 실행 파일 (통합 시스템)
- `config.json`: 설정 파일
- `requirements.txt`: 필요한 패키지 목록

## 동작 원리
1. ARP 스캔으로 네트워크 상의 기기 목록 확인
2. 설정된 신뢰할 수 있는 기기(사용자 기기)의 존재 여부 확인
3. 웹캠을 통해 실시간으로 사람 감지
4. 사용자 기기가 없는 상태에서 사람이 감지되면 침입으로 판단
5. 경보 발생 (콘솔 알림 및 향후 푸시 알림 구현 예정)

## 주의사항
- ARP 스캔은 관리자 권한이 필요할 수 있습니다 (Windows에서는 관리자 권한으로 실행)
- 방화벽 설정에 따라 ARP 스캔이 제한될 수 있습니다
